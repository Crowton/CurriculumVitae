\cvevent{}{Space Efficient Functional Off-line Partial Persistent Trees\newline with Applications to Planar Point Location}
{2022 \textit{Accepted at WADS'23}}{}

{\small

Authors: G. S. Brodal, C. M. Rysgaard, J. K. R. Schou, and R. Svenning.

% \vspace{4pt}

% Abstract: Sarnak and Tarjan presented a simple slab based algorithm for planar point location using partially persistent search trees.
% It allows for an elegant, purely functional implementation with construction time $\Oh{n\log n}$, supporting point location queries in time $\Oh{\log n}$, and using space $\Oh{n\log n}$. A non-functional solution  (using the \textit{fat node} and \textit{node copying} technique of Driscoll et al.) allows the space to be reduced to $\Oh{n}$. In this paper we present the first purely functional solution matching the previous time bounds and using space $\Oh{n}$. To do so we present a general technique for transforming an ephemeral, linked data structure into an off-line, partially persistent, purely functional data structure with additive $\Oh{n\log n}$ construction time and $\Oh{n}$ space overhead; for a series of $n$ updates.
